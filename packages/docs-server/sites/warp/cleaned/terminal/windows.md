 1. [Global Hotkey](/terminal/windows/global-hotkey) is a configurable shortcut that can show/hide a dedicated window or all windows on your chosen desktop regardless of whether the app is focused.
 2. [Tabs](/terminal/windows/tabs) allow you to organize a window into multiple terminal sessions.
 3. [Split Panes](/terminal/windows/split-panes) allows you to divide a Tab into multiple rectangular _panes_ , each of which is a unique terminal session.
