##### Do you need information about other SDKs?
LaunchDarkly does not offer SDKs for all languages or frameworks. If you’re searching for information about using Svelte, Preact, or Angular with LaunchDarkly, consider if you can use the JavaScript SDK instead. To learn more, read [JavaScript SDK reference](/docs/sdk/client-side/javascript).
To request support for a specific language or framework, [start a Support ticket](https://support.launchdarkly.com/hc/en-us/requests/new).
