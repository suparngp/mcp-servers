##### Relay Proxy Enterprise is available to customers on select plans
Relay Proxy Enterprise is only available to customers on select plans. To learn more, [read about our pricing](https://launchdarkly.com/pricing/). To upgrade your plan, [contact Sales](https://launchdarkly.com/contact-sales/).
## Overview
This section explains the features that are available for Enterprise customers who use the Relay Proxy.
## Relay Proxy Enterprise features
The core set of features in the Relay Proxy is available to all customers on all plans. To learn more about what features are available to all customers, read [The Relay Proxy](/docs/sdk/relay-proxy).
Enterprise and Guardian customers can take advantage of additional features that make the Relay Proxy more secure and faster to configure:
 * [Automatic configuration](/docs/sdk/relay-proxy/automatic-configuration)
 * [Offline mode](/docs/sdk/relay-proxy/offline)
We recommend that all customers on Enterprise and Guardian plans who are primarily feature flagging in server-side applications use Relay Proxy Enterprise. To learn more about the costs and benefits of using the Relay Proxy, read [Relay Proxy use cases](/docs/sdk/relay-proxy/use-cases).
