 * [React SDKs](#react-sdks)
LaunchDarkly has two distinct React SDKs. The React Web SDK works on web-based React codebases, and the React Native SDK is built to work on native apps for Android and iOS. To learn more about each SDK, read the [React Web SDK reference](/docs/sdk/client-side/react/react-web) and the [React Native SDK reference](/docs/sdk/client-side/react/react-native).
Additionally, we have a plugin for Gatsby-based websites. The Gatsby plugin wraps the React Web SDK and makes it easier to use in a Gatsby site.
