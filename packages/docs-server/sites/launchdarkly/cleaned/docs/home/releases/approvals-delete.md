## Overview
This topic explains how to delete approval requests.
## Delete approval requests
You may delete your request for changes at any point before the changes are applied. You can do this in several ways:
 * from the **Approvals** dashboard
 * from the approval request
 * for approval requests for flag or AI Config changes, from the flag or AI Config’s “Pending changes” panel
![](https://fern-image-hosting.s3.us-east-1.amazonaws.com/launchdarkly/openapi-logo.svg)
You can also use the REST API:
[Delete approval request](/docs/api/approvals/delete-approval-request)
## Delete approval requests from the Approvals dashboard
To delete an approval request from the Approvals dashboard:
 1. Click **Approvals** in the left sidenav. The **Approvals** dashboard opens.
 2. Find the request you wish to delete and click the **overflow menu**.
![The overflow menu with the "Delete request" option called out.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/5ba127dfaca43c8259d02220a0c75fdbcb2096ed04f6cd38423e0be014ade6e1/assets/images/auto/approvals-delete-request-dashboard.auto.png)
The overflow menu with the "Delete request" option called out.
 3. Choose “Delete request.” A confirmation dialog appears.
 4. Click **Delete**. The request is deleted.
## Delete an approval request from the “Pending changes” panel
To delete approval requests from a flag or AI Config’s “Pending changes” panel:
 1. Navigate to the flag or AI Config and click the “Pending changes” section in the right sidebar, or the **checkmark** pending changes icon. The “Pending changes” panel appears.
 2. Find the change you wish to delete and click the **overflow menu**. Additional options appear:
![The "Delete" option in the overflow menu.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/b347ed8f01f963854569e469ccd8a8a2603f83958f7495c237c1c5f09e56f2d5/assets/images/auto/workflows-pending-changes-overflow.auto.png)
The "Delete" option in the overflow menu.
 3. Choose **Delete**. A confirmation dialog appears.
 4. Click **Delete**. The request is deleted.
## Delete approval requests from the request details screen
To delete an approval request from the request details screen:
 1. Navigate to the request details screen from the **Approvals** dashboard, flag, AI Config, segment, or approval notification.
 2. Click the **three-dot** overflow menu and choose “Delete request.” A confirmation dialog appears.
![The overflow menu expanded with the "Delete request" option.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/5ba127dfaca43c8259d02220a0c75fdbcb2096ed04f6cd38423e0be014ade6e1/assets/images/auto/approvals-delete-request-dashboard.auto.png)
The overflow menu expanded with the "Delete request" option.
 1. Click **Delete request**. The request is deleted.
## Delete an approval request from the “Pending changes” panel
This procedure only applies to approval requests for flag and AI Config changes.
To delete approval requests from a flag or AI Config’s “Pending changes” panel:
 1. Navigate to the flag or AI Config and click the “Pending changes” section in the right sidebar, or the **checkmark** pending changes icon. The “Pending changes” panel appears.
 2. Find the change you wish to delete and click the **overflow menu**. Additional options appear:
![The "Delete" option in the overflow menu.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/b347ed8f01f963854569e469ccd8a8a2603f83958f7495c237c1c5f09e56f2d5/assets/images/auto/workflows-pending-changes-overflow.auto.png)
The "Delete" option in the overflow menu.
 3. Choose **Delete**. A confirmation dialog appears.
 4. Click **Delete**. The request is deleted.
