## Overview
This topic explains how to use tags with LaunchDarkly resources. Tags are customizable labels that help you categorize flags, segments, metrics, projects, environments, and other resources.
Tags are useful in the following ways:
 * Categorizing: You can add the same tag to different kinds of resources. This lets you easily understand which resources are associated with which projects or teams.
 * Filtering: You can filter the **Flags** list, environments list, and projects list by tag.
 * Permissions: You can allow or deny account members access to resources with specific tags. To learn how to use tags with custom role policies, read [Using tags](/docs/home/account/roles/role-tags).
You can add tags to a resource when you create the resource, or you can add tags to existing resources later. Tag names can include only letters, numbers, periods `.`, underscores `_`, and dashes `-`. Tags are not case-sensitive. After you create a tag, you cannot delete it.
The following sections explain how to add tags to existing resources.
## Flags
To add tags to a flag:
 1. Navigate to the **Flags** list.
 2. Click on the name of the flag you want to add tags to. The flag’s **Targeting** tab appears.
 3. Click on the **pencil** icon in the “Tags” section of the right sidebar. The tags list appears.
 4. Create a new tag or select a tag from the list.
To learn how to add tags to flags when you create them, read [Creating new flags](/docs/home/flags/new).
![](https://fern-image-hosting.s3.us-east-1.amazonaws.com/launchdarkly/openapi-logo.svg)
You can also use the REST API: [Update feature flag](/docs/api/feature-flags/patch-feature-flag)
To remove a tag from a flag, click on the tag’s **x** in the right sidebar.
### Filter the Flags list by tag
You can filter the **Flags** list by one or more tags.
To filter your flags:
 1. Click **Filters**.
 2. Select **Add tag(s)**.
 3. Search for or select a tag from the list.
Your **Flags** list now displays only the flags that match the filter you specified.
## Segments
To add tags to a segment:
 1. Navigate to the **Segments** list.
 2. Click on the name of the segment you want to add tags to. The segment’s **Targeting** page appears.
 3. Click on the **pencil** icon in the “Tags” section of the right sidebar. The “Choose tag(s)” list appears.
 4. Create a new tag or select a tag from the list.
To learn how to add tags to segments when you create them, read [Creating segments](/docs/home/flags/segments-create).
![](https://fern-image-hosting.s3.us-east-1.amazonaws.com/launchdarkly/openapi-logo.svg)
You can also use the REST API: [Patch segment](/docs/api/segments/patch-segment)
To remove a tag from a segment:
 1. Click on the **gear** icon in the segment’s right sidebar. The “Segment settings” page appears.
 2. Click on the **x** for the tag you want to remove.
 3. Click **Save**.
## Metrics
To add tags to a metric:
 1. Navigate to the **Metrics** list.
 2. Click on the name of the metric you want to add tags to. The metric details page appears.
 3. Click on the **pencil** icon in the “Tags” section of the right sidebar. The “Choose tag(s)” list appears.
 4. Create a new tag or select a tag from the list.
To learn how to add tags to metrics when you create them, read [Metrics](/docs/home/metrics).
![](https://fern-image-hosting.s3.us-east-1.amazonaws.com/launchdarkly/openapi-logo.svg)
You can also use the REST API: [Patch metric](/docs/api/metrics/patch-metric)
To remove a tag from a metric, click on the tag’s **x** in the right sidebar.
## Projects
To add tags to a project:
 1. Click the project dropdown. The project menu appears:
![The project menu.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/e7170a6ea679fd8c8210aa3660093eb18394effdc9571d713b907cbce33e24c0/assets/images/auto/project-menu-dropdown.auto.png)
The project menu.
 1. Select **Project settings**. The **General** list appears.
 2. Under Tags, click **Add tags**.
 3. A menu with the list of tags appears.
 * To add existing tags to your project, check the checkbox next to one or more tags in the list.
 * To add a new tag to your project, type the new tag into the search box. Then check the checkbox next to “Create new tag.”
To learn how to add tags to projects when you create them, read [Create projects](/docs/home/account/project#create-projects).
![](https://fern-image-hosting.s3.us-east-1.amazonaws.com/launchdarkly/openapi-logo.svg)
You can also use the REST API: [Update project](/docs/api/projects/patch-project)
To remove a tag from a project:
 1. Click the project dropdown. The project menu appears.
 2. Select **Project settings**. The **General** list appears.
 3. In the “Tags” section, click on the tag’s **x**.
### Filter the projects list by tag
You can filter the projects list by tag.
To filter your projects:
 1. Click the **gear** icon in the left sidenav to view Organization settings.
 2. Click **Projects**.
 3. Select a tag to filter by from the **Tags** menu.
Your projects list now displays only the projects that match the tag you specified.
## Environments
To add tags to an environment:
 1. Click the project dropdown. The project menu appears:
![The project menu.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/e7170a6ea679fd8c8210aa3660093eb18394effdc9571d713b907cbce33e24c0/assets/images/auto/project-menu-dropdown.auto.png)
The project menu.
 1. Select **Project settings**.
 2. Select **Environments**. The **Environments** list appears.
 3. Next to the environment you want to add tags to, click the **three-dot** overflow menu.
 4. Select **Edit environment**.
 5. Select or create a new **Tag**.
 6. Click **Save environment**.
To learn how to add tags to environments when you create them, read [Create environments](/docs/home/account/environment/create).
![](https://fern-image-hosting.s3.us-east-1.amazonaws.com/launchdarkly/openapi-logo.svg)
You can also use the REST API: [Update environment](/docs/api/environments/patch-environment)
To remove a tag from an environment:
 1. Click the project dropdown. The project menu appears.
 2. Select **Project settings**.
 3. Select **Environments**. The **Environments** list appears.
 4. Next to the environment you want to add tags to, click the **three-dot** overflow menu.
 5. Select **Edit environment**.
 6. In the **Tags** field, click on the tag’s **x**.
 7. Click **Save environment**.
### Filter the environments list by tag
You can filter an environments list by tag.
To filter your environments:
 1. Click the project dropdown. The project menu appears:
![The project menu.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/e7170a6ea679fd8c8210aa3660093eb18394effdc9571d713b907cbce33e24c0/assets/images/auto/project-menu-dropdown.auto.png)
The project menu.
 1. Select **Project settings**.
 2. Select **Environments**. The **Environments** list appears.
 3. Select a tag to filter by from the **Tags** menu.
Your environments list now displays only the environments that match the tag you specified.
## Experiments
To add tags to an experiment or A/A test:
 1. Navigate to the **Experiments** list.
 2. Click on the name of the experiment you want to add tags to. Depending on whether or not the experiment is running, the experiment’s **Design** or **Results** tab appears.
 3. Click on the **pencil** icon in the “Tags” section of the right sidebar. The tags list appears.
 4. Create a new tag or select a tag from the list.
To remove a tag from an experiment, click on the tag’s **x** in the right sidebar.
### Filter the Experiments list by tag
You can filter the **Experiments** list by one or more tags.
To filter your experiments:
 1. Click **Filters**.
 2. Select **Add tag(s)**.
 3. Search for or select a tag from the list.
The **Experiments** list now displays only the experiments that match the filter you specified.
## Multi-armed bandits
To add tags to a multi-armed bandit (MAB):
 1. Navigate to the **Multi-armed bandits** list.
 2. Click on the name of the MAB you want to add tags to. Depending on if the MAB is running or not, the MAB’s **Design** or **Results** tab appears.
 3. Click on the **pencil** icon in the “Tags” section of the right sidebar. The tags list appears.
 4. Create a new tag or select a tag from the list.
To remove a tag from a MAB, click on the tag’s **x** in the right sidebar.
### Filter the Multi-armed bandits list by tag
You can filter the **Multi-armed bandits** list by one or more tags.
To filter your experiments:
 1. Click **Filters**.
 2. Select **Add tag(s)**.
 3. Search for or select a tag from the list.
The **Multi-armed bandits** list now displays only the MABs that match the filter you specified.
## Delete tags
To delete a tag from your list of tags, you must first remove it from every resource in your account. LaunchDarkly will then automatically delete the tag.
