## Overview
This topic describes project settings and how to update them.
## Available project settings
The following table describes the various project settings that you can update:
Project settings | Changes you can make 
---|--- 
**General** | 
 * Update the project’s **Name**.
 * Update the project’s [tags](/docs/home/account/tags#projects).
 * [Delete the project](/docs/home/account/project#delete-projects).
**Views** | 
 * Create, edit, or delete views in this project. To learn more, read [Views](/docs/home/account/views).
**Environments** | 
 * Create, edit, or delete environments in this project. To learn more, read [Environments](/docs/home/account/environment).
**Flag settings** | 
 * Update the [Client-side availability](/docs/home/flags/proj-flag-settings#client-side-availability) of flags in this project.
 * Update the [Flag key conventions](/docs/home/flags/proj-flag-settings#flag-key-conventions) for this project.
 * Update the [Flag templates](/docs/home/flags/proj-flag-settings#flag-templates) for flags in this project
**Release pipelines** | 
 * Create, edit, or delete release pipelines in this project. To learn more, read [Release pipelines](/docs/home/releases/release-pipelines).
**Context kinds** | 
 * Create, edit, or delete context kinds in this project. To learn more, read [Context kinds](/docs/home/flags/context-kinds).
**AI model configs** | 
 * Create and delete AI model configurations in this project. To learn more, read [Create AI model configurations](/docs/home/ai-configs/create-model-config).
**Approval settings** | 
 * Require approvals for changes to AI Configs. To learn more, read [AI Config approvals](/docs/home/releases/approval-config#ai-config-approvals).
**Observability** | 
 * Update settings for reviewing sessions and errors. Add ingestion filters for sessions, errors, logs, and traces. To learn more, read [Observability settings](/docs/home/observability/settings).
**Lifecycle settings** | 
 * Customize the criteria that LaunchDarkly uses to determine when flags are ready to be archived. To learn more, read [Flag lifecycle settings](/docs/home/flags/flag-lifecycle-settings).
## Edit projects
 1. Click the project dropdown. The project menu appears:
![The project menu.](https://files.buildwithfern.com/https://launchdarkly.docs.buildwithfern.com/docs/e7170a6ea679fd8c8210aa3660093eb18394effdc9571d713b907cbce33e24c0/assets/images/auto/project-menu-dropdown.auto.png)
The project menu.
 1. Select **Project settings**.
 2. The **General** list of project settings appears.
From here, you can update the project’s name or tags. You can also [delete the project](/docs/home/account/project#delete-projects).
To make other changes, select the appropriate settings page from the list. [Available project settings](/docs/home/account/edit-project#available-project-settings) are described above.
