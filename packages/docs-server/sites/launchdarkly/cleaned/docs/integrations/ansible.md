Development work on the Ansible Collection integration is ongoing. Elements of this integration may change without notice.
## Overview
This topic links to the documentation for the LaunchDarkly Ansible Collection integration. Ansible Collection is an automation tool you can configure to manage some tasks in your LaunchDarkly account, such as creating projects and flags or syncing data across environments.
To learn more, read the [Ansible Collection integration documentation](https://launchdarkly-labs.github.io/ansible-launchdarkly-collection/).
