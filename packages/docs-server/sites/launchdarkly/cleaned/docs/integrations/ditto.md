## Overview
This topic links to Ditto’s documentation for its legacy LaunchDarkly integration. Ditto is a platform that serves as a single source of truth for product copy, managing text from draft to design to development. You can sync Ditto variants in your workspace with LaunchDarkly feature flags.
Ditto manages this integration. To learn more, read Ditto’s [LaunchDarkly Integration](https://dittov3.notion.site/LaunchDarkly-Integration-256cc8865c7a80ed8512f39772dae8a8).
