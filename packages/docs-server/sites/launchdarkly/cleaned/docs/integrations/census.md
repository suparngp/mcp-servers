##### The Census integration is available to customers on select plans
The Census integration is only available to customers on select plans. To learn more, [read about our pricing](https://launchdarkly.com/pricing/). To upgrade your plan, [contact Sales](https://launchdarkly.com/contact-sales/).
## Overview
This topic links to Census’ documentation for its LaunchDarkly integration. Census is a data activation tool you can use to send metric events from any supported data source to LaunchDarkly. You must be using LaunchDarkly’s Experimentation product to integrate with Census. To learn more, read [Experimentation](/docs/home/experimentation) and [Metrics](/docs/home/metrics).
Census manages this integration. To learn how to use it, read Census’ [LaunchDarkly documentation](https://docs.getcensus.com/destinations/launchdarkly).
