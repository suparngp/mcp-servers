##### The Cortex integration is available to customers on select plans
The Cortex integration is only available to customers on select plans. To learn more, [read about our pricing](https://launchdarkly.com/pricing/). To upgrade your plan, [contact Sales](https://launchdarkly.com/contact-sales/).
## Overview
This topic links to Cortex’s documentation for its LaunchDarkly integration. With the integration, you can view and monitor feature flags in the service catalog. You can also write Scorecard rules to enforce feature flag usage in Cortex and drive operational maturity.
Cortex manages this integration. To learn more, read Cortex’s [LaunchDarkly documentation](https://docs.cortex.io/docs/reference/integrations/launchdarkly).
