## Overview
This topic links to Bitrise’s documentation for its LaunchDarkly integration. Bitrise Release Management is a platform for managing mobile app releases on the App Store and Google Play. With this integration, you can monitor the status of both app store releases and feature flags from a single location.
Bitrise manages this integration. To learn more, read Bitrise’s [Integrating LaunchDarkly feature flags](https://devcenter.bitrise.io/en/release-management/configuring-connected-apps/integrating-launchdarkly-feature-flags.html).
