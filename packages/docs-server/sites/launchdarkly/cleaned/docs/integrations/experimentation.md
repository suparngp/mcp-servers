## Overview
This category lists integrations from third-party apps related to Experimentation and metrics, and must be used in conjunction with Experimentation or metric features. To learn more, read [Experimentation](/docs/home/experimentation) and [Metrics](/docs/home/metrics).
LaunchDarkly supports the following Experimentation and metric integrations:
 * [Census](/docs/integrations/census)
 * [Segment](/docs/integrations/metric-segment)
 * [Sentry](/docs/integrations/sentry)
 * [Snowplow](/docs/integrations/snowplow)
 * [Sprig](/docs/integrations/sprig)
 * [Tealium](/docs/integrations/tealium)
