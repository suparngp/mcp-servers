## Overview
Convex is a sync platform and framework that replaces your backend and client state management. Using the LaunchDarkly integration, you can sync your LaunchDarkly environment to an application you’ve developed on Convex, allowing you to instantly install feature flagging in your TypeScript or JavaScript applications.
Convex manages this integration. To learn how to use it, read Convex’s [LaunchDarkly Feature Flags](https://www.convex.dev/components/launchdarkly).
