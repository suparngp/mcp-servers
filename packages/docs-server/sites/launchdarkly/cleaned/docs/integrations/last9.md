##### The Last9 integration is available to customers on select plans
The Last9 integration is only available to customers on select plans. To learn more, [read about our pricing](https://launchdarkly.com/pricing/). To upgrade your plan, [contact Sales](https://launchdarkly.com/contact-sales/).
## Overview
Last9 Levitate’s integration with LaunchDarkly enables change intelligence with feature flag events. You can use the integration to visualize feature flag events as Change Event annotations in Levitate to correlate system health, and to dynamically update alert configs to automate alert management and reduce noise.
Last9 manages this integration. To learn how to use it, read Last9’s [LaunchDarkly Integration](https://docs.last9.io/docs/levitate-integrations-launchdarkly).
