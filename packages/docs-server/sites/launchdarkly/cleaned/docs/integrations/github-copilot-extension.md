##### The LaunchDarkly extension for GitHub Copilot is available to customers on select plans
The LaunchDarkly extension for GitHub Copilot on Visual Studio Marketplace is available only in conjunction with the LaunchDarkly VSCode extension. The VSCode extension is only available to customers on select plans. To learn more, [read about our pricing](https://launchdarkly.com/pricing/).
