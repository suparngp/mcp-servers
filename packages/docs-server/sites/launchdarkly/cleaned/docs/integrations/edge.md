## Overview
This category lists integrations with third-party edge providers. Each integration requires that you use the appropriate edge SDK. To learn more, read [Edge SDKs](/docs/sdk/edge).
LaunchDarkly supports the following edge tools integrations:
 * [Akamai](/docs/integrations/akamai)
 * [Cloudflare](/docs/integrations/cloudflare)
 * [Fastly](/docs/integrations/fastly)
 * [Netlify](/docs/integrations/netlify)
 * [Vercel](/docs/integrations/vercel)
