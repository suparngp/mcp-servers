## Overview
This category lists integrations from third-party apps related to collaboration.
##### Some integrations are available to customers on select plans
Some integrations are only only available to customers on select plans. To learn more, [read about our pricing](https://launchdarkly.com/pricing/). To upgrade your plan, [contact Sales](https://launchdarkly.com/contact-sales/).
LaunchDarkly supports the following collaboration integrations:
 * [Confluence](/docs/integrations/confluence)
 * [Microsoft Teams](/docs/integrations/microsoft-teams)
 * [Slack app](/docs/integrations/slack)
