## Overview
This category lists Environments as a Service (EaaS) integrations from third-party apps.
If you are a potential LaunchDarkly partner interested in building your own EaaS integration, read [Building an ephemeral Environments as a Service integration](/docs/guides/integrations/build-eaas).
LaunchDarkly supports the following EaaS connector integrations:
 * [Okteto](/docs/integrations/okteto)
 * [Release](/docs/integrations/release)
 * [Roost.ai](/docs/integrations/roost-ai)
