##### The Osano integration is available to customers on select plans
The Osano integration is only available to customers on select plans. To learn more, [read about our pricing](https://launchdarkly.com/pricing/). To upgrade your plan, [contact Sales](https://launchdarkly.com/contact-sales/).
## Overview
This topic links to Osano’s documentation for its LaunchDarkly integration.
Osano discovers and automatically classifies personally identifiable information (PII) stored within context attributes in LaunchDarkly. When you receive a subject rights request Osano will search context attributes for PII, such as names and email addresses.
Osano manages this integration. To learn how to use it, read Osano’s [LaunchDarkly Integration](https://developers.osano.com/integrations/data-discovery-integrations/launchdarkly).
